# win11-dotfiles

My latest Windows rice inspired from Selenized theme. See more screenshots from [here](assets/screenshots/)

![](assets/screenshots/02.png)
![](assets/screenshots/03.png)

## File Structure (more for my own reference)

```
~/.config/
├── cava
├── fastfetch
├── komorebi
├── yasb
├── starship.toml
├── whkdrc

~/AppData/Local
├── nvim

~/AppData/Roaming
├── bottom
    ├── bottom.toml    
├── FlowLauncher
    ├── Themes
        ├── SelenizedDark.xaml
├── spicetify
    ├── Themes
        ├── text
            ├── color.ini
            ├── user.css
```
